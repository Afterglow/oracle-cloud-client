# Oracle Cloud API Client

`oracle-cloud` is a RubyGem for interacting with the Oracle Cloud REST API.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'oracle-cloud'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install oracle-cloud
